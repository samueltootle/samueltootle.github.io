bco_utils::update_adapted_field

extrapolate variable fields to domains that are/should be zero based on the coefficients on the boundary, e.g:

  • extrapolate space-time fields into the horizon of the BH from outside the horizon
  • extrapolate matter fields from inside the star to outside of the star

Note 1:
This is important before importing fields from one numerical space to another as interpolating the fields to the new
space may have problems if the adjacent domain is zero. After import, remember to set relevant domains back to zero
Note 2:
could be generalized to Tensors. Currently only done for Scalars

Template Parameters
dom_ttype of domain.
Parameters
[input]res: reference to modifiable Scalar field that contains the current solution to the field. This will be changed!
[input]from_dom: index of domain to extrapolate FROM
[input]to_dom: index of domain to extrapolate TO
[input]dom: pointer to domain to extrapolate TO
[input]bound: which boundary we extrapolating from (OUTER_BC, INNER_BC)
[output]res: Modified scalar field with values interpolated from from_dom to to_dom
