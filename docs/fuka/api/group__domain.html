Detailed Description

Each domain class are derived from the abstract class Domain. The domains contain the informations on the number of collocation points and of coefficients and, more importantly define the coordinates use by implementing the relations between the numerical coordinates and the physical ones.

The class Space is an ensemble of Domains describing the computational space.

update_fields

this function updates the constant fields used in the generation of various ID solvers. Since the fields are a function of the cartesian basis, they must be updated every iteration as the coordinates change. However, since they are a "constant" in the solver, they must be updated manually. This includes the fields themselves and their value within the system of equation. See update_field()
