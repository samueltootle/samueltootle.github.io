Detailed Description

The class template Array implements mult-dimensional arrays (mainly of integers and doubles). The size of the various dimensions are stored in the class Dim_array and the position in a given array by the class Index.

The class Matrice is used for various matrices operations, mainly to solve linear systems.

Param is the storage class.

All these classes are independent of the numerical method (spectral method).
