interpolate_radial

We interpolate radially in 3D - f(r, theta, phi) - for an arbitrary excision surface for a given field before extrapolating inside the excision surface

[input] space: numerical space [input] field_in: field to interpolate on [input] order_: Interpolation order [input] dr_: spacing of points to use in interpolation [input] offset_: offset from excision to start interpolation [input] r_: radius to evaluate [input] r_bound_: initial guess of boundary radius [input] theta_: angle from z to xy plane [0, pi] [input] phi_: angle inside xy plane [0, 2pi] [input] dom_: dom just outside excision surface [input] xshift_: coordinate x shift (for binaries)
