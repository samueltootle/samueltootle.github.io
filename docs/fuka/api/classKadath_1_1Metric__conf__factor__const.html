Detailed Description

  • Class to deal with a metric with a conformal decomposition (mainly used for AADS spacetimes) The true metric and the conformal are related via . The conformal factor vanishes at some boundary so that the various quantities (Christoffels) are multiplied by appropriate factors of $\Omega$ to ensure regularity. The metric is assumed to be constant.
