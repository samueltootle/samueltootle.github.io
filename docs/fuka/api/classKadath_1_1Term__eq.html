Detailed Description

This class is intended to describe the manage objects appearing in the equations. Basically a Term_eq can be either a double or a tensorial field. It is also defined only in a given Domain and not in the whole space. ALong with the quantity itself it also contains its variation that is used by the automatic differentiation algorithm to compute the Jacobian of the system.
