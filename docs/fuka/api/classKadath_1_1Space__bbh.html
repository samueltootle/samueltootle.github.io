Detailed Description

Spacetime intended for binary black hole configurations in full general relativity (see constructor for details about the domains used) The shape of the horizons are variables.

Standard constructor

Parameters
ttype[input] : the type of basis.
dist[input] : distance between the centers of the two spheres.
r1[input] : radius of the first hole (assumed spherical at this point).
r2[input] : radius of the second hole (assumed spherical at this point).
rbi[input] : radius of the outer boundary of the bispherical part.
rext[input] : outer radius of the.
nr[input] : number of points in each dimension

The various domains are then :

  • One Domain_shell_outer_adapted around the first hole.
  • One Domain_shell_inner_adapted around the first hole.
  • One Domain_shell_outer_adapted around the second hole.
  • One Domain_shell_inner_adapted around the second hole.
  • One Domain_bispheric_chi_first near the first hole.
  • One Domain_bispheric_rect near the first hole.
  • One Domain_bispheric_eta_first inbetween the two holes.
  • One Domain_bispheric_rect near the second hole.
  • One Domain_bispheric_chi_first near the second hole.
  • One Domain_shell centered on the origin, ecnompassing the two holes.
