Detailed Description

Class for a spherical-like domain, having a symmetry with respect to the plane . It is two-dimensional, intended for fields having a sylmmetry with respect to .

  • 2 dimensions.
  • centered on the point center

The outer surface is spherical whereas the inner boundary is deformed (typically to accomodate things like surfaces of stars).
