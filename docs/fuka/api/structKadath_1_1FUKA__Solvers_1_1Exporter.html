Detailed Description

template<class config_t, class space_t>
struct Kadath::FUKA_Solvers::Exporter< config_t, space_t >

The following "Exporter" is really a bandage until the FUKA_Solvers can be rewritten. Essentially many of the utilities her are duplicate to the solvers, however, pointers are used instead of reference thereby allowing the possibility for dynamic allocation for, e.g. multi-threaded importing of the initial data. It would have no impact on the solving of the system of equations.
