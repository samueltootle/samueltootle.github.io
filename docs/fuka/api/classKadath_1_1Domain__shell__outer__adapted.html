Detailed Description

Class for a spherical-like domain, having a symmetry with respect to the plane .

  • 3 dimensions.
  • centered on the point center

The inner surface is spherical whereas the outer boundary is deformed (typically to accomodate things like surfaces of stars).

The link between numerical and physical coordinates is as follows :

is the fixed inner boundary and is an angular function giving the value of the outer variable boundary.
